# macOS Port

This is a port of the original Windows game to macOS, built with Swift and SpriteKit.

The port is being developed with the assistance of [Claude Code](https://claude.ai/code), Anthropic's AI coding tool, which is helping with the translation from C# to Swift, debugging, and adapting the original SDL-based rendering and input systems to native macOS APIs.

## Status

Work in progress — actively being ported and debugged.
